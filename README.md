# estimate-contract-app

社会保険労務士齊藤事務所向けの見積書・契約書作成アプリです。

現在は GitHub Pages で公開できるように、単体 HTML 版を `index.html` として配置しています。

## 現在の機能

- 顧問契約の月次報酬見積書作成
- 業務委託契約書作成
- 料金計算明細の出力
- 月次報酬一覧と手続き料金表の出力
- スポット手続きの見積書作成
- スポット手続きのご請求書作成
- スポット明細の追加、削除、数量変更
- 一覧外手続きの手入力追加
- ブラウザ印刷による PDF 保存

GitHub Pages 公開後、公開 URL にアクセスして利用します。

PDF 出力はブラウザの印刷機能を使用します。
